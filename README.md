## 이 레포지토리는 뭔가요?

Google AdMob 광고 인증을 위한 app-ads.txt 파일을 호스팅하는 용도입니다.

GitHub Pages를 이용해 mycms2007.github.io 도메인을 만들고,
AdMob이 이 주소를 크롤링해서 광고 게재 권한을 확인합니다.

- 앱: 화투 짝맞추기 (Android)
- 광고 플랫폼: Google AdMob
- 파일 위치: https://mycms2007.github.io/app-ads.txt
